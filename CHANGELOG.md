# Changelog

## 1.0.0

- Terminal GTK3 + VTE (lekki, zgodny z motywami Cinnamon/GTK)
- Tryb hasła z maskowaniem `*` i możliwością cofania (overlay)
- Obsługa myszy: menu kontekstowe (Kopiuj/Wklej) + wklejanie środkowym przyciskiem
- Multi-lang (PL/EN) na bazie `LANG` + `VISTULA_LANG`
- Ikona aplikacji `org.vistula.Terminal` (SVG)
